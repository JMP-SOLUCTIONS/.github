## 🧾 OneClick ERP

Contamos con nuestro software empresarial **OneClick ERP**, un excelente sistema de facturación, inventario y control administrativo diseñado para todo tipo de rubro comercial.

**OneClick ERP** permite a las empresas gestionar sus operaciones de forma ordenada, rápida y segura, integrando diferentes módulos en una sola plataforma.

### Módulos principales

- Facturación
- Inventarios
- Nómina y Recursos Humanos
- Compras
- Contabilidad
- Control administrativo

## 🎟️ SmartTurn

También contamos con **SmartTurn**, un sistema de gestión de turnos que permite organizar la atención al cliente mediante tickets, pantalla de llamado, código QR y control de filas.

Este sistema ayuda a evitar filas largas, mejora la experiencia del cliente y permite una atención más ordenada.

- Cooperativas

- Clínicas
- Farmacias
- Barberías
